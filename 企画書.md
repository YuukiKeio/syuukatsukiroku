# 就活キロク（syukatsukiroku）企画書

## 1. 项目概要

**一句话**：面向在日本求职（就活）的学生，用来记录选考进度、日程、笔记与结果的个人 Web 应用。

| 项目 | 内容 |
|---|---|
| 名称 | 就活キロク（syukatsukiroku） |
| 语言 | 日语 UI |
| 线上地址 | https://syukatsukiroku.vercel.app |
| 代码仓库 | https://github.com/YuukiKeio/syuukatsukiroku |
| 参考 | NEXTROUND2（简洁白底+强调色的卡片风格） |

**设计目标**：功能优先、界面清爽、隐私安全（数据全部存本地）、手机上单手可用。

---

## 2. 目标用户与场景

- **用户**：正在进行就活的学生（尤其投递数量多、选考环节复杂的行业：金融・商社・コンサル 等）
- **核心痛点**：
  - 同时进行多家公司、多个选考环节，容易混乱、错过日程
  - ES 设问、面接问答、GD、说明会笔记散落各处
  - 想回顾"投了几家、通过几家、哪类还没做完"
- **典型场景**：说明会现场记笔记、面接前复习 ES 和自己的回答、每天早上确认今天的日程

---

## 3. 技术方案

| 层 | 选型 | 理由 |
|---|---|---|
| 前端 | 原生 HTML / CSS / JavaScript（无框架） | 单文件即可运行，零依赖，加载快 |
| 数据存储 | 浏览器 localStorage | 无需后端、无需账号，隐私性好 |
| 托管 | Vercel（连 GitHub 自动部署） | push 即自动更新，免费 |
| 备份 | JSON 导出/导入 | 弥补 localStorage 不能跨设备的缺点 |

**文件结构**（重构后）：
- `index.html` — 页面结构（169 行）
- `style.css` — 主题变量与样式（约 440 行）
- `app.js` — 全部逻辑（约 1100 行，按模块分区）

## 4. 功能清单

### 4.1 首页
- **🔔 今日・明日 バナー**：置顶显示今明两天的所有事件，点击直达详情
- **搜索框**：按 会社名 / 種類 / 活動名 实时过滤
- **三大板块**：進行中 / これからの予定 / 振り返り，按最近日程自动排序
- **同名企业合并**：同一公司的多个项目合并成一张卡（📁 N件），点开进入企业总览页
- **卡片信息**：种类彩色标签、日程、场所、步骤进度、紧迫角标（あと2日）、通过标记、FB 标识

### 4.2 新規追加（右上 ＋）
- 会社名（输入 2 字起自动提示已有公司名）
- 活動名（如「1day仕事体験」「夏季5daysコース」）
- 種類（见下）＋ 自定义
- 日程（Apple 日历式：開始→終了，默认今天，支持单日/跨天、时间段）
- 場所（オンライン=贴URL / 対面=贴地址 / なし）
- ステータス

### 4.3 種類（彩色分类）
| 颜色 | 种类 |
|---|---|
| 🔵 蓝 | インターン・夏インターン・冬インターン・ワークショップ・イベント |
| 🟢 绿 | 説明会・セミナー・オープンカンパニー |
| 🔴 红 | OB・OG訪問 |
| 🟣 紫 | 本選考 |
| ⚪ 灰 | 自定义种类 |

### 4.4 详情页（选考记录核心）
- **状态切换**：進行中 / 予定 / 振り返り
- **✏️ 編集**：改名字、活動名、种类（改名时マイページ情报自动跟随）
- **日程・场所**：可随时编辑，含地图/链接快捷入口
- **結果标记**：ES通過 / 参加決定
- **选考步骤**（可逐个添加，各带结构）：
  | ステップ | 内容 |
  |---|---|
  | ES | 設問＋回答（多个，回答实时字数统计） |
  | 動画視聴 / 説明会 / GD / インターン | ノート |
  | 動画選考 | 質問＋ノート |
  | 面接 | ノート＋質問（每题可标 予測/本番） |
- 每个步骤有 ✅ 完成勾选
- **📝 振り返り**：整体感想笔记
- **💬 フィードバック**：可选，记录企业/面试官反馈
- **🔐 マイページ情報**：URL / ログインID / パスワード（密码可显隐，按公司共享）
- **📅 iOSカレンダーに登録**：导出 .ics

### 4.5 統計（可折叠，状态记忆）
- 数字卡：全エントリー / ES通過 / 参加決定 / ステップ完了率
- 種類別の内訳（条形图）
- ステップ別の件数（蓝=总数，绿=已完成）

### 4.6 カレンダー
- 月历，事件按种类同色显示
- 跨天活动连续显示
- 点日期看当天列表，点事件跳详情

### 4.7 系统功能
- **自动保存**：每次输入即时写入 localStorage，切后台/关闭时兜底保存
- **⚙ 备份**：JSON 导出 / 导入
- **iOS 日历导出**：新建带日程后自动询问，或详情页手动导出 .ics
- **🌙 暗色模式**：一键切换，跟随系统，记忆选择

## 5. 数据结构（localStorage）

```
{
  entries: [{
    id, name, type, subtitle, status,           // 基本信息
    startDate, endDate, time, endTime,           // 日程
    locType, locValue,                           // 场所
    marks: [],                                   // ES通過/参加決定
    review, hasFeedback, feedback,               // 振り返り・FB
    steps: [{
      id, kind, date, time, note, done,
      questions: [{ id, q, a, tag }]             // ES设问/面接问答
    }]
  }],
  customTypes: [],                               // 自定义种类
  companies: { "会社名": { mypageUrl, loginId, password } }  // 按公司共享
}
```

- 存储键：`syuukatsu-kiroku-v1`（迄今未变，向后兼容）
- 主题键：`syuukatsu-theme`；统计折叠键：`syuukatsu-stats-collapsed`

## 6. 隐私与安全

- 所有数据仅存于**当前浏览器本地**，不上传任何服务器、不关联个人身份
- 换设备/浏览器不互通 → 提供 JSON 备份导入导出
- マイページ密码为明文本地存储，界面提示「共用PCでは入力しないでください」
